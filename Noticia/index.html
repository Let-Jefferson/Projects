O que é o CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e o layout de documentos HTML e XML. Foi desenvolvida pela World Wide Web Consortium (W3C) e lançada pela primeira vez em 1996 como uma forma de separar a estrutura do conteúdo da sua apresentação.

Antes do CSS

Nos primórdios da web, os estilos eram aplicados diretamente nas tags HTML, resultando em código desorganizado e difícil de manter. Em 1994, Håkon Wium Lie propôs a separação entre estrutura e estilo, dando origem à ideia do CSS.

Um novo jeito de estilizar

A primeira versão oficial do CSS, conhecida como CSS1, foi lançada em 1996. Ela introduziu a sintaxe básica do CSS e permitiu a definição de estilos para cores, fontes, margens e outros aspectos de apresentação.

Em 1998, foi lançado o CSS2, que trouxe muitos recursos novos, como posicionamento avançado, layout em várias colunas e suporte para impressão. O CSS2 também introduziu seletores mais complexos e pseudo-classes.

Linguagem mais moderna

A partir de 1999, o desenvolvimento do CSS3 começou a adicionar uma série de módulos independentes, cada um trazendo novos recursos e capacidades. Esses módulos incluíam seletores avançados, efeitos de transição e animação, sombras, bordas arredondadas, fontes personalizadas e muito mais. Diversos módulos do CSS3 foram implementados pelos navegadores ao longo do tempo.

Embora não exista uma especificação CSS4 completa, a evolução do CSS continua com a criação de novos módulos CSS e aprimoramentos dos recursos existentes. As implementações modernas do CSS são frequentemente referidas como CSS3, mesmo que incluam recursos além da especificação original do CSS3.

Atualmente, não existe uma especificação oficial do CSS5. O desenvolvimento do CSS evoluiu com a adoção do CSS3, que introduziu módulos independentes para adicionar recursos e funcionalidades ao longo do tempo.

A abordagem adotada pela comunidade de desenvolvimento web tem sido a de trabalhar em módulos separados, em vez de lançar uma versão completa do CSS5. Esses módulos abrangem várias áreas, como layout, tipografia, animações, transformações, entre outros.

Os módulos do CSS3, como Flexbox, Grid, Transitions, Animations, Font Face, Backgrounds, Box Shadows, entre outros, foram implementados pelos navegadores ao longo do tempo, e muitos desses recursos são amplamente suportados pelos navegadores modernos.

A evolução do CSS continua com a criação de novos módulos e aprimoramentos dos recursos existentes. Os trabalhos na especificação do CSS continuam sob a supervisão da World Wide Web Consortium (W3C), a organização responsável por desenvolver os padrões da web.

Embora o termo "CSS5" possa ser usado informalmente para se referir às futuras adições e aprimoramentos no CSS, é importante ter em mente que o desenvolvimento do CSS é contínuo e incremental, com novos recursos sendo introduzidos em módulos individuais, em vez de uma versão completa do CSS5.
